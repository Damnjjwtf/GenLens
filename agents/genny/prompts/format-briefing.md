# Format GenLens Briefing

Write for working creative technologists, not AI hobbyists.

Use:

- Clear vertical sections.
- Strong signal titles.
- One-line production-impact summaries.
- Metric chips only when source-backed.
- Source links.
- Article preview images when available.

Avoid:

- Corporate tone.
- Hype without numbers.
- Paltry sample-sized briefs.
- Local-only HTML as the final artifact.
- Old Schultz or VibeCopy naming.
