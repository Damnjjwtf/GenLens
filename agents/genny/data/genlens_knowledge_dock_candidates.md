## Use Policy

This file is not canonical truth.

Genny may use it to generate research questions, source targets, role hypotheses, product ideas, glossary terms, and editorial frames. Genny must not publish specific claims, rankings, dates, legal statements, salary figures, cost benchmarks, or model status updates from this file unless they are verified against primary or high-trust sources.

## Useful Product Structure To Adopt

### Directed Generation

Useful framing:

- The visual AI market is moving from prompt novelty to directed generation.
- Production users care about camera control, character consistency, reference clips, shot continuity, and repeatability.
- The relevant distinction is not "text-to-video" but "can I direct the output like a controllable production system?"

Genny action:

- Score visual AI signals higher when they mention control primitives, reference workflows, shot continuity, editability, or deterministic iteration.

### Production ROI Benchmarks

Useful framing:

- GenLens should track cost/time deltas between traditional production and AI-native production.
- ROI language should be specific: cost per minute, time per 60-second video, cost per image, VFX task reduction, latency, usable-output rate.

Genny action:

- Treat ROI numbers as publishable only when attached to a verifiable source.
- If a number is unverified, call it a claim and keep it in the research queue.

### Model/Tool Matrix

Useful framing:

- Professional users need model-to-task routing, not a single "best model."
- Genny should map each model/tool to a production use case:
  - drafting volume.
  - hero finishing.
  - image-to-video iteration.
  - physics/motion.
  - character consistency.
  - enterprise compliance.
  - open-source restyling.

Genny action:

- Build tool stack blueprints around task fit, not leaderboard rank.
- Verify any leaderboard/ranking claim before publishing.

### Signal Brief Template

Useful template:

- Platform / Technology.
- Core Dynamic.
- Concrete Use Case.
- Strategic Impact.
- Source Verification Link.

Genny action:

- This matches `genlens_signal_schema.md`; use it as the required card structure.

### Rights, Provenance, Compliance

Useful framing:

- Compliance is a core creative-production signal, not a legal afterthought.
- Track consent, likeness, labeling, C2PA/content credentials, union terms, dataset rights, and commercial indemnification.

Genny action:

- Prioritize official sources for legal/compliance claims:
  - SAG-AFTRA.
  - C2PA.
  - EU official sites.
  - vendor legal/product docs.
  - Adobe Content Credentials / CAI docs.

### Build This

Useful framing:

- Genny should convert signals into weekend-scoped proof builds.
- Proof builds should measure latency, fidelity, workflow repeatability, cost, and handoff quality.

Candidate proof-build themes:

- Repeatable game-to-video node.
- Synthetic lip-sync audit.
- Product physics lab.
- AI shot pipeline proof.
- Creative inference ops dashboard.

### Anti-Slop / Substance Moat

Useful framing:

- Professional value comes from control, craft, repeatability, and source-backed claims.
- "Slop" should mean loss of control, generic aesthetics, unverifiable claims, and low-trust production.

Genny action:

- Reject stories that only celebrate visual novelty without control, workflow, rights, or measurable production impact.

### Monetization / Authority Model

Useful framing:

- GenLens can become an authority layer for creative AI.
- Potential products:
  - source quality dashboard.
  - authority/citation audits.
  - role arbitrage reports.
  - tool stack blueprints.
  - hiring briefs.
  - proof-build generator.
  - partner citation gap audits.

Genny action:

- Treat these as product hypotheses until validated by actual customer pull or source data.

When a user asks Genny to use this dock:

3. Promote verified claims into a signal, source entry, tool manifest update, or role radar note.
